Projeto realizado durante evento IEL 4.I no IFSC-Lages em 2022.

O objetivo do projeto foi desenvolver um software para anáçise de custos de usinagem e 
encontrar o ponto de maior eficiência economica para a operação ao considerar curto de 
energia, desgate de ferramenta e salário do operador.

Dependencias:
tkinter
matplotlib

Excute o comando abaixo no diretório raiz:
python usinamais.py
